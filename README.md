# Gastronot

Gastronaut is a local food ordering application. With this application, you can order food in the local restaurants of your city.

## Used technologies
- Android Projesi
- Mvvm
- Dagger Hilt
- Kotlin Coroutine
- ViewModel - Livedata
- Retrofit - Gson
- RecyclerView
- Fragment
- Socket for live support
- Firebase
